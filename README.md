# Titanic-Machine_Learning_from_Disaster
O desafio do Titanic, promovido pelo Kaggle, é o “Hello, World!” da Ciência de Dados. Usando dados reais do naufrágio de 1912, o objetivo é analisar fatores que influenciaram a sobrevivência dos passageiros e criar modelos preditivos que estimem suas chances de sobreviver à tragédia.
